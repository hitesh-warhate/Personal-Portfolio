# Hitesh Warhate — Developer Portfolio

A futuristic, developer-focused 3D portfolio website built with **React**, **TanStack Start**, **Three.js**, and **Tailwind CSS**. Showcases my work in **Agentic AI**, **LLM systems**, and **full-stack development**.

🌐 **Live:** [your-deployed-url-here]

---

## ✨ Features

- 🎨 **Interactive 3D Hero Scene** — wireframe icosahedrons, orbit rings, and floating code ribbons built with `@react-three/fiber` and `@react-three/drei`
- 💻 **Developer-first aesthetic** — terminal-inspired UI, mono typography, code-window profile card, and section numbering (`// 01 — ABOUT`)
- 🌌 **Trust-building palette** — deep blue & cyan tokens defined in `oklch` for accurate, accessible color
- ⚡ **Fully responsive** — tuned from mobile to ultrawide
- 🧩 **Sectioned layout** — Hero, About, Stack, Projects (10 GitHub repos), Contact
- 🔍 **SEO ready** — semantic HTML, meta tags, single H1, descriptive titles
- 🚀 **SSR via TanStack Start** with Vite 7 + React 19
